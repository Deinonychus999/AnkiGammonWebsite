Features

AnkiGammon is designed for flexibility and ease of use. Customize every aspect of your study cards while enjoying seamless workflow automation.

Intelligent Blunder Filtering

Drag XG files (.xg) onto the window and AnkiGammon automatically keeps only blunders above your error threshold. Focus on meaningful mistakes, skip trivial decisions. Match file (.mat) support coming soon with automatic GnuBG analysis and filtering!

Smart Format Detection

Paste any format: XGID, OGID, GNUID position IDs, or full XG analysis text. AnkiGammon automatically detects the format and provides helpful warnings if dependencies are needed.

Flexible Board Customization

Choose from 6 beautiful color schemes (Classic, Forest, Ocean, Desert, Sunset, Midnight) and toggle board orientation (clockwise/counter-clockwise). All settings persist automatically.

Automatic GnuBG Analysis

Paste position IDs without analysis and AnkiGammon automatically runs GnuBG to generate complete move and cube analysis. Configure analysis depth (0-4 ply) to balance speed and accuracy.

Smart Import Filtering

Set your error threshold (default 0.080) to automatically filter game records. Only import positions with significant blunders. Choose specific players to study. Your filter settings persist between sessions.

Interactive Move Visualization

Click moves on the card back to see animated checker movements with smooth arc paths. Optional feature you can enable/disable to match your learning preference.

Dual Export Methods

Choose AnkiConnect for instant one-click export to running Anki, or APKG files for offline/portable decks. Visual progress tracking shows real-time status with cancellation support.

Score Matrix Generation

For cube decisions, optionally generate detailed score matrices showing equity at different match scores. Essential for understanding cube actions in tournament play.

Complete Analysis

Every card includes equity values, win/gammon/backgammon percentages, and error calculations for all candidate moves. Customize whether options appear on the card front for easier or harder study.

Frequently Asked Questions

Do I need XG analysis or can I just paste position IDs?

You can paste just position IDs (XGID, OGID, or GNUID format) without any analysis. AnkiGammon automatically runs GnuBG in the background to generate complete move and cube analysis. You can configure the analysis depth (0-4 ply) in settings. For cube decisions, it can even generate full score matrices showing equity at different match scores.

Can I filter which positions get imported from game files?

Yes! This is one of AnkiGammon's most powerful features. When importing .xg files, you can filter by:

  • Error threshold: Set a minimum error threshold (default 0.080) to automatically import only significant blunders. Skip trivial decisions and focus on positions where you made meaningful mistakes.
  • Player selection: Choose to import mistakes from Player X (White), Player O (Black), or both. Perfect for studying only your own mistakes from a game record.

These settings are remembered between sessions, so you don't need to reconfigure them each time. Coming soon: Match file (.mat) support with automatic GnuBG analysis of non-analyzed matches and intelligent blunder filtering. Just drag and drop!

What customization options are available for the flashcards?

AnkiGammon offers extensive customization options, all automatically saved:

  • Color schemes: 6 themes (Classic, Forest, Ocean, Desert, Sunset, Midnight)
  • Board orientation: Counter-clockwise (standard) or clockwise numbering
  • Show options toggle: Display multiple-choice options on card front (easier) or hide them (harder)
  • Interactive moves: Enable/disable animated checker movements when clicking moves on the card back
  • Score matrix generation: Optionally generate detailed cube equity matrices for tournament positions
  • GnuBG analysis depth: Choose 0-4 ply to balance speed vs. accuracy
  • Export method: AnkiConnect (instant) or APKG (portable files)

Do I need to reconfigure settings every time I open the app?

No! All settings automatically persist to ~/.ankigammon/config.json. Your color scheme, board orientation, deck name, export method, GnuBG path, analysis depth, import filters, and all other preferences are saved immediately when changed. The next time you open AnkiGammon, everything loads exactly as you left it. Even your window size and position are remembered.

What is GNU Backgammon (GnuBG) and how do I install it?

What is GnuBG?
GNU Backgammon is a free, world-class backgammon engine that plays and analyzes at expert level. AnkiGammon uses it to automatically analyze positions when you paste just position IDs without analysis. It's completely optional but enables the automatic analysis feature.

Installation:

  • Windows: Download the installer (version 1.07) from ftp.gnu.org and run the .exe file.
  • macOS: Use Homebrew: brew install --cask gnubg
  • Linux: Install from your package manager: sudo apt install gnubg (Ubuntu/Debian) or sudo dnf install gnubg (Fedora)

Note: macOS and Linux package managers may install a different version than the Windows installer.

Once installed, AnkiGammon will automatically detect and use GnuBG for position analysis. You can configure the analysis depth (0-4 ply) in the settings.

Will this mess up my existing Anki decks?

No. AnkiGammon creates a separate deck (default name: "My AnkiGammon Deck") with no interference to existing cards. You can customize the deck name in settings or by clicking the edit button next to the deck name.

What's the difference between AnkiConnect and APKG export?

AnkiConnect (recommended): Cards appear instantly in Anki while it's running. No import step needed. Requires AnkiConnect addon. Best for daily workflow.

APKG export: Creates a portable .apkg file you manually import into Anki. Useful when Anki isn't running or for sharing decks with others. Your export method preference is saved.

Can I customize the board appearance?

Yes. AnkiGammon offers 6 color schemes to match your preference: Classic (traditional brown/beige), Forest (green/brown), Ocean (blue/teal), Desert (tan/orange), Sunset (purple/orange), and Midnight (dark blue/purple). You can also toggle board orientation between counter-clockwise (point 1 at bottom right, standard) and clockwise (point 1 at bottom left). All preferences are saved automatically and applied to all generated cards.
