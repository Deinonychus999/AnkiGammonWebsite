What is .xg?

                    
The .xg format is the native binary format used by eXtreme Gammon, a popular backgammon analysis program. It stores full match records with analysis data.

                

                

                    
What is .mat?

                    
The .mat format (Jellyfish format) is a widely-supported plain text format for backgammon matches. Compatible with GNU Backgammon, Backgammon Studio, and many other tools.
